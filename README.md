# Celloct

`Celloct`는 직원 근무시간 관리 및 재고관리 시스템이 없는 소상공인을 위한 프로젝트입니다.

## 구성

### Backend

- SpringBoot 2.6.4
- DataBase
  - real: mysql
  - local: h2
- Spring Security
- Spring Data JPA

### Frontend

- thymeleaf
- flutter (예정)
